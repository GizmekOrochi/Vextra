Detailed Description

Abstract base class for all Vextra CLI commands.

This class defines the unified interface for every command that the Vextra CLI tool can execute (e.g. create, build, help).

Design goals

  • Avoid string copies by using std::string_view for lightweight command identifiers.
  • Provide a consistent, extensible command interface.
  • Enable dynamic command registration and dispatching via CommandManager.

Pattern

Implements the Command Pattern, where each subclass encapsulates a specific operation and can be executed interchangeably.
